# 🛠️ Smart Hardware Bypass & Override System

A real-world hardware engineering project designed to create an automatic frequency modulator and safety bypass circuit for vehicle horn networks. This system combines physical toggle overrides with programmable relay switching.

---

## ⚙️ Project Overview & Logic
* **Manual Mode (Bypass):** The circuit completely disconnects the microcontroller and routes power directly to the horn, acting as a standard vehicle network.
* **Automatic Mode (Override):** Routes power through a 5V Relay module controlled by an Arduino, creating customized pulse intervals and warning frequencies.

---

## 📦 Components Required
* **Arduino Uno / Nano** (The automation brain)
* **5V Single-Channel Relay Module** (To handle the vehicle current safely)
* **SPDT Toggle Switch** (For physical bypass selection)
* **Connecting Wires & 12V Power Source / Battery**
